# 修复插件安装状态显示错误

## Goal
修复插件安装页面/流程中实例 t13 明明已经挂掉，但界面仍显示为运行中的问题。

## Requirements
- 定位安装插件时实例运行状态的数据来源与展示逻辑
- 复现实例已停止但前端仍显示运行中的问题
- 修复状态判定或刷新逻辑，确保已挂掉的实例不会继续显示为运行中
- 补充必要的测试或验证步骤，覆盖该场景

## Acceptance Criteria
- [ ] 能稳定复现或通过代码路径证明问题原因
- [ ] t13 进程退出/挂掉后，页面不会继续显示为运行中
- [ ] 安装插件相关已有流程不受影响
- [ ] 完成至少一次针对该场景的自测验证

## Technical Notes
可能涉及前端状态轮询、后端任务状态缓存、安装流程状态汇总逻辑，先调研后确定修改范围。
